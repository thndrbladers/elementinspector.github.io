Frequently Asked Questions

How do I report a bug?

Send an email to elementinspector.help@gmail.com with "Bug Report" in the subject, or create an issue on our GitHub repository. Please include browser version, extension version, and steps to reproduce.

Can I request new features?

Absolutely! We welcome feature requests. Send an email or use GitHub issues to suggest improvements. Include use cases and expected behavior.

Is Element Inspector open source?

Yes! Element Inspector is open source under the MIT license. You can view, fork, and contribute to the code on GitHub.

How can I contribute?

Contributions are welcome! You can contribute code, documentation, translations, or bug reports. Check our GitHub repository for contribution guidelines.

Do you offer technical support?

Yes, we provide support for technical issues. Use the contact form or email us directly. We typically respond within 24-48 hours.

Can I use Element Inspector commercially?

Yes! Element Inspector is free for personal and commercial use under the MIT license. No attribution required, but appreciated.
