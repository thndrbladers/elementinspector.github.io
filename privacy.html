Overview

Element Inspector is a privacy-first Chrome extension designed for web developers, designers, and QA professionals. This Privacy Policy explains how we handle information when you use our extension.

🔒 Privacy-First Approach

Element Inspector operates entirely locally on your device. We do not collect, store, or transmit any personal data or browsing information to external servers.

Data Collection

What We DON'T Collect

  • Personal information (name, email, phone number)
  • Browsing history or website visits
  • Page content or analyzed elements
  • IP addresses or location data
  • Device information or browser fingerprinting
  • Usage analytics or behavioral tracking
  • Cookies or web beacons
  • User credentials or authentication data

What We DO Store Locally

  • Tool Preferences: Your selected analysis tools (stored in browser local storage)
  • Tab-Specific States: Temporary tool states per browser tab (cleared when tab closes)
  • Extension Settings: UI preferences and configuration options

Important: All data storage is local to your device. Nothing is transmitted to external servers.

🔑 Permissions Explained

The extension requires minimal permissions for core functionality:

Required Permissions

activeTab: Access to the currently active tab to analyze page elements
Why needed: To inject inspection tools and read element properties
Data accessed: Page DOM structure (not stored or transmitted)
scripting: Inject content scripts to enable element inspection
Why needed: To overlay analysis tools on web pages
Data accessed: None - only injects functionality
storage: Store user preferences locally
Why needed: Remember your tool selections between sessions
Data stored: Tool preferences only (no personal data)
tabs: Manage tool states across different browser tabs
Why needed: Maintain separate tool states per tab
Data accessed: Tab identification only (no content)

Optional Permissions

Currently, no optional permissions are requested.

How We Use Data

The data processed by Element Inspector is used exclusively for:

  • Displaying real-time element measurements and properties
  • Providing color analysis and extraction tools
  • Showing alignment and spacing information
  • Generating visual overlays and inspection guides

All processing is temporary and data is not stored permanently on your device.

Data Sharing

✅ Zero Data Sharing

Element Inspector does not share, sell, rent, or distribute any data to third parties. Since we don't collect personal data, there's nothing to share.

What We DO NOT Use

  • Analytics services (Google Analytics, etc.)
  • Crash reporting services
  • Advertisement networks
  • Social media integrations
  • Content delivery networks for tracking
  • External APIs for data processing
  • Cloud storage services

External Resources

  • Chrome Web Store (for updates only)
  • Extension icons (bundled with extension, no external loading)

Data Storage

Local Storage Only

Element Inspector uses minimal local storage for:

  • User Preferences: Your tool selection and UI settings
  • Extension State: Which tools are currently active

Data Retention

  • Inspection data is processed in real-time and not stored
  • User preferences persist until you uninstall the extension
  • No browsing history or website data is retained

Your Rights

Since Element Inspector doesn't collect personal data, traditional data protection rights don't apply. However, you have full control over:

🛠️ Extension Control

Enable, disable, or uninstall the extension at any time through Chrome's extension management.

⚙️ Settings Management

Configure which tools are active and customize the inspection interface.

🗑️ Data Removal

All local data is automatically removed when you uninstall the extension.

🔍 Transparency

Full transparency in our privacy practices and data handling.

Your Privacy Rights

  • Right to Information: This privacy policy provides complete transparency
  • Right of Access: View your data via browser developer tools
  • Right to Rectification: Modify settings directly in the extension
  • Right to Erasure: Clear all data via browser extension settings
  • Right to Data Portability: Export via browser's extension management
  • Right to Object: Uninstall extension to stop all data processing

❓ Frequently Asked Questions

Q: Can you see what websites I visit?

A: No. The extension only activates when you manually enable it on a specific tab.

Q: Do you track how I use the extension?

A: No. We don't implement any analytics or usage tracking.

Q: Is my analyzed data stored anywhere?

A: No. All analysis happens in real-time and nothing is saved.

Q: Can other extensions see Element Inspector's data?

A: No. Each extension operates in its own isolated environment.

Q: What happens if I uninstall the extension?

A: All local data (tool preferences) is automatically deleted.

Data Retention

  • Persistent Settings: Tool preferences stored until you clear browser data
  • Temporary States: Tab-specific states cleared when tabs are closed
  • Manual Deletion: All data can be cleared via browser extension settings

Cross-Platform Considerations

  • Extension updates delivered through Chrome Web Store
  • No personal data transmitted during update process
